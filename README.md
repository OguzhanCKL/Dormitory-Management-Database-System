# Dormitory Management Database System

## 📌 Overview
This project is a robust relational database solution designed to automate and manage the daily operations of a student dormitory. It centralizes information for accommodation, staff operations, maintenance, and financial processes.

---

## 🔑 Key Entities & Features
* **Student Management:** Tracks personal info, academic details, and residency status.
* **Room & Capacity Tracking:** Manages occupancy rates and real-time room availability.
* **Financial Auditing:** Tracks student payments, billing history, and staff salaries.
* **Maintenance Operations:** System for reporting and tracking facility maintenance tasks.
* **Staff Coordination:** Manages staff roles, shifts, and departmental assignments.

---

## 🛠️ Technical Details
* **RDBMS:** SQL (Oracle / PostgreSQL compatible).
* **Database Design:** Fully normalized relational schema (3NF/BCNF) to ensure data integrity.
* **Implementation:** Includes complex queries, constraints, and triggers for data validation.
